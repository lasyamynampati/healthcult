# Security and DPDP Baseline

## Mandatory controls
- Explicit user consent capture for each processing purpose.
- Purpose limitation tags in audit logs and data flows.
- Role-based access control with least privilege.
- Encryption in transit and at rest for PHI-like data.
- Retention policies with deletion workflows and evidence logs.

## Operational controls
- Login and privileged action audit trails.
- Queue and API abuse protections (rate limits, retry guardrails).
- Incident response and data subject request workflows in runbooks.
